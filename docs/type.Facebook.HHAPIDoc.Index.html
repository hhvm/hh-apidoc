Facebook\HHAPIDoc\Index

An index of all the Documentables that we should act on

This can be used for auto-linking, generating index documented, or other purposes

namespace Facebook\HHAPIDoc;

type Index = shape(
  'types' => keyset<string>,
  'newtypes' => keyset<string>,
  'functions' => keyset<string>,
  'classes' => dict<string, keyset<string>>,
  'interfaces' => dict<string, keyset<string>>,
  'traits' => dict<string, keyset<string>>,
);

Fields

  • 'types': keyset<string> - A list of all transparent type aliases
  • 'newtypes': keyset<string> - A list of all opaque type aliases
  • 'functions': keyset<string> - A list of all function names
  • 'classes': dict<string, keyset<string>> - A map from class names to a list of method names
  • 'interfaces': dict<string, keyset<string>> - A map from interfaces names to a list of method names
  • 'traits': dict<string, keyset<string>> - A map from trait names to a list of method names
