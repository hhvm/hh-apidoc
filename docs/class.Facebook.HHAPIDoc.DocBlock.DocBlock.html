Facebook\HHAPIDoc\DocBlock\DocBlock

Class to represent and parse a docblock (a

k.a. doc comment).

These comments always are delimeted by /** and *\/.

DocBlocks are treated as GitHub Flavored Markdown, and standard JavaDoc style tags are supported, such as @param, @see, and @return.
